## Breakpoint

Renders children when breakpoint requirements are met

<Playground>
  <Breakpoint only="phone">
    <Text>Only phone</Text>
  </Breakpoint>
  <Breakpoint down="tablet">
    <Text>Tablet & Phone</Text>
  </Breakpoint>
  <Breakpoint up="desktop">
    <Text>Desktop</Text>
  </Breakpoint>
</Playground>

## BreakpointProvider

Returns current breakpoint

<Playground>
  <BreakpointProvider>
    {({breakpoint, isMobile}) => (
      <Row>
        <Col mr={2}>
          <Text>{breakpoint}</Text>
        </Col>
        <Col>
          <Text color={isMobile ? 'pink' : 'grey'}>({isMobile ? 'is mobile' : 'not mobile'})</Text>
        </Col>
      </Row>
    )}
  </BreakpointProvider>
</Playground>

## withBreakpoint

Wraps the target component in a `BreakpointProvider` and passes the resulting `breakpoint` to props
